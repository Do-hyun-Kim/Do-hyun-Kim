## Introduce

- 사용자에게 다양한 삶을 제공할 수 있다는 것이 매력적이라 생각한 iOS 개발자 김도현입니다.
- iOS 개발자는 사용자의 일상에 손쉽게 접근 할 수 있도록 도메인을 제공하는 개발자이며, 이를 통해 다양한 기능을 제공하는 개발자라 생각합니다.
